Help

About

This web application is used for storing and presenting meta data for your multimedia files (such as images or movies).

The following meta data can be added to each file from the file collection:

  • File description
  • File date and time
  • Persons
  • Locations
  • Tags

The meta data above, called categories, can be used when searching for files or to document historic events represented by the files.

Your Files

FileDB does not modify any files in your file collection.

Before importing your files, take a moment to think about the directory structure. Any directory structure can be used for sorting files in your existing collection. However, this layout has been proven to work very well:

root
 |- 2015
 |   |- 2015-06-12 - Biking
 |   |      |- file1.jpg
 |   |      |- file2.jpg
 |   |
 |   |- 2015-07-13 - Birthday party
 |          |- somefile.jpg
 |
 |- 2016
     |- 2016-01-05 - Something else
            |- anotherfile1.jpg
            |- anotherfile2.jpg

Files can have the same name in different directories.
