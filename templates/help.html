Help

About

This web application is typically used for adding meta data to a collection of multimedia files (such as images or movies). The meta data is called categories in this web application.

File Categories

The following meta data can be added to each file from the file collection:

  • Person
  • Location
  • Tag

File Collection

Any directory structure can be used for sorting files in your existing collection. However, this layout has been proven to work very well:

root
 |
 |- 2015
 |   |- 2015-06-12 - Biking
 |          |- file1.jpg
 |          |- file2.jpg
 |
 |- 2016
     |- 2016-01-05 - Something else
            |- anotherfile1.jpg
            |- anotherfile2.jpg
The root directory can be placed anywhere, see configuration chapter.

Configuration

Configuration parameters are set in Python code (see config.py).

API

This web application fetches and updates data in the FileDB database via its API. The API can be used by other applications.
